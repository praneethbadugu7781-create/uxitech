# Quick Start Guide - UXI Mobile App

## 1️⃣ First Time Setup

```bash
cd mobile-app
npm install
```

## 2️⃣ Run the App

```bash
npm start
```

Then press:
- **`a`** → Android emulator
- **`i`** → iOS simulator (Mac only)
- **`w`** → Web preview

## 3️⃣ Common Commands

| Command | Purpose |
|---------|---------|
| `npm start` | Start dev server |
| `npm run android` | Build & run on Android |
| `npm run ios` | Build & run on iOS (Mac) |
| `npm run web` | Run on web browser |
| `npm run prebuild` | Generate native folders |
