## 🚀 Quick Start

### Installation
```bash
cd mobile-app
npm install
```

### Run Development
```bash
npm start

# Then press:
# a - Android emulator
# i - iOS simulator
# w - Web preview
```

### Build Production
```bash
# Android
npm run android

# iOS (Mac only)
npm run ios

# Web
npm run web
```

## ❓ FAQ

### Q: Can I run on my phone?
**A:** Yes! Use Expo Go app to scan QR code from `npm start`

### Q: How do I change the app icon?
**A:** Replace `assets/expo.icon/` with your icon (1024x1024 PNG)

### Q: How do I customize colors?
**A:** Edit the hex color values in each screen's `styles` object

### Q: Can I add more screens?
**A:** Yes! Create new files in `app/(tabs)/` and add to `_layout.tsx`

### Q: How do I add a chatbot?
**A:** Create a new screen or modal with chatbot UI

### Q: Is it ready for production?
**A:** Yes! Just needs icons/splash screen customization before app store upload
